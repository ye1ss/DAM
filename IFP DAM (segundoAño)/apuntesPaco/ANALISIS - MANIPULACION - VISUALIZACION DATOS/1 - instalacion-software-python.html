2. Distribución Anaconda¶

La manera ms sencilla de instalar todo lo que necesitaremos es usar la distribucin Anaconda.

Como podis ver en el siguiente esquema, Anaconda es una distribucin de cdigo abierto que agrupa los paquetes ms importantes para hacer minera de datos en Python. Incluye el gestor de paquetes conda y un conjunto de libreras de data science, junto con los Python Notebooks.

En las prximas actividades, usaremos principalmente las libreras pandas y numpy para hacer minera de datos, y matplotlib para la visualisacin.

2.1 Instalación de la distribución¶

Escoged vuestro sistema operativo y seguid las instrucciones para instalar la distribuin Anaconda.

2.1.1 Instalación en Linux¶

  1. Ir a https://www.anaconda.com/download/#linux y bajaros la versin 3.6 de la distribucin Anaconda.

  2. Abrir el terminal y ejecutar el comando bash ~/Downloads/Anaconda3-5.0.1-Linux-x86_64.sh. Si el fichero se llama de otra forma, adaptar el comando a vuestra situacin.

  3. El instalador os pedir permiso para mostrar la licencia: "In order to continue the installation process, please review the license agreement. Click Enter to view license terms". Aceptr con la tecla Enter

  4. Leer la licencia, o hacer scroll hasta el final, y escribir Yes para aceptarla.

  5. El instalador os pedir el directorio donde lo queris instalar. Si aceptis, el directorio por defecto ser /home/<user>/anaconda3
  6. Ahora os pedir si queris aadir el path de Anaconda en el .bashrc. Decirle que s.
  7. Ejecutar el comando source ~/.bashrc para finalizar la instalacin.
  8. Tambin en el terminal, ejecutar el comando anaconda-navigator. Se os obrir un navegador con la aplicacin Anaconda.

2.1.2 Instalación en Windows¶

  1. Ir a https://www.anaconda.com/download/#windows y bajaros la versin 3.6 de la distribucin Anaconda.
  2. Hacer doble-clic en el instalador.
  3. Clicar Siguiente.
  4. Aceptar la licencia.
  5. Seleccionar Solo yo y clicar Siguiente.
  6. Seleccionar la carpeta de instalacin y clicar Siguiente.
  7. A continuacin os saldr un dilogo con la opcin de aadir Anaconda como variable de entorno en el PATH. No seleccionar esta opcin. Seleccionar la opcin de usar Anaconda como la versin por defecto de Python 3.6. Clicar Siguiente.
  8. Clicar el botn Instalar y esperar hasta que se acabe la instalacin.
  9. Podis abrir la aplicacin desde el men de inicio de Windows.

2.1.3 Instalación en macOS¶

  1. Ir a https://www.anaconda.com/download/#macos y bajaros la versin 3.6 de la distribucin Anaconda.
  2. Hcer doble-clic en el instalador.
  3. Pasar las pantallas de la Introduccin, el Readme y la Licencia.
  4. Seleccionar Instalar solo para mi y clicar Continuar.
  5. Clicar el botn Instalar y esperar hasta que se acabe la instalacin.
  6. Podis abrir la aplicacin desde el Launchpad.

2.2 Anaconda Navigator¶

El Anaconda Navigator es una interfcie grfica de usuario (GUI) que permite ejecutar las aplicaciones incluidas en la distribucin e instalar paquetes sin tener que usar la lnia de comandos.
