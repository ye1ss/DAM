-------------------------------- CONTROLES DEL PLAYER -----------------------------------------------------

WASD/Flechas/Mando_Joystick -> Movimiento
Space/Mando_A -> Saltar
E/Mando_Y -> Interacción con objetos
J/Mando_X -> Hacha
K/Mando_LB -> Daga
L/Mando_RB -> Guadaña

Escape/Mando_B -> Atrás-Menú
Mouse1/Mando_A -> Aceptar-Menú
